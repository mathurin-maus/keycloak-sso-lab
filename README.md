# Keycloak SSO Lab

## Overview

This project is a personal lab environment created to learn and experiment with Identity and Access Management (IAM) and Single Sign-On (SSO) concepts using Keycloak.

The lab was built while preparing the implementation of a Keycloak-based SSO solution in a professional environment.

## Environment

- Rocky Linux
- Docker
- Keycloak
- Nginx

## Objectives

- Understand Keycloak architecture
- Configure authentication and authorization
- Implement Single Sign-On (SSO)
- Connect multiple applications to the same Identity Provider
- Explore IAM concepts

## Current Architecture

Rocky Linux VM

├── Docker

│ ├── Keycloak

│ ├── Application 1 (Nginx)

│ └── Application 2 (Coming soon)
